# TravelSearcher 개발 로드맵 (ROADMAP.md)

본 문서는 주요 항공권 예약 플랫폼(Skyscanner, Trip.com, 네이버 항공권)과의 비교 분석을 바탕으로, `TravelSearcher`의 경쟁력을 높이고 'Zero Maintenance Cost(사용자 및 관리자 관리 비용 제로)' 목표를 달성하기 위한 향후 개발 방향을 제시합니다.

---

## 1. 주요 경쟁 서비스 벤치마킹 분석

| 플랫폼 | 핵심 강점 (Key Features) | TravelSearcher 적용점 |
| :--- | :--- | :--- |
| **Skyscanner** | 전 세계 항공사/여행사 검색, 'Everywhere' 최저가 탐색, 월별 최저가 그래프 | 현재 구현된 '어디든지' 기능을 전 세계 지도 기반 탐색으로 확장 가능 |
| **Trip.com** | 앱 내 직접 예약 시스템, 24/7 고객 지원, 호텔/액티비티 통합 패키지 | 예약 딥링크(Deep Link) 강화 및 여정 관리 자동화 필요 |
| **네이버 항공권** | 국내 LCC 연동 최적화, **국내 카드사별 프로모션 할인**, 다구간 검색 | 임직원 전용 카드/혜택 연동 및 다구간(Multi-city) 검색 지원 필요 |

---

## 2. 현재 프로젝트 진단 (`TravelSearcher`)

- **강점**: Next.js 16 기반의 세련된 UI/UX, 반응형 디자인(Drawer 활용), 실시간 위치 자동 완성, 가격 추이 시각화.
- **약점**: 실제 예약 흐름 부재, 상세 정보(수하물 등) 누락, 필터 로직의 완전한 백엔드 연동 필요, 임직원 인증 및 특가 정보 부족.

---

## 3. Top 5 Missing Features (우선순위 순)

사용자 편의성과 '관리 비용 제로' 목표를 위해 우선적으로 도입해야 할 5가지 핵심 기능입니다.

### ① 다구간(Multi-city) 여정 검색 지원
- **이유**: 단순 왕복/편도를 넘어 복잡한 여행 경로를 설계하려는 프리미엄 사용자 니즈 충족.
- **기능**: A -> B, C -> D 와 같은 복잡한 연결 여정 검색.

### ② 가격 변동 추적 및 자동 알림 (Price Alert)
- **이유**: 'Zero Maintenance' 달성을 위해 사용자가 매번 검색할 필요 없이, 원하는 조건 만족 시 푸시/메일 전송.
- **기능**: 특정 노선/날짜의 가격 하락 시 자동 알림.

### ③ 임직원 특가 및 카드 프로모션 자동 연동
- **이유**: '임직원 전용 서비스'라는 정체성 강화 및 네이버 항공권 수준의 체감 할인율 제공.
- **기능**: 로그인된 임직원 계정에 따른 특별 할인율 자동 계산 및 카드사 혜택 표기.

### ④ 항공권 상세 정보 및 수하물 규정 시각화
- **이유**: 예약 결정 단계에서 가장 중요한 정보인 수하물 포함 여부, 좌석 간격, 기종 정보 제공.
- **기능**: FlightCard 확장 시 상세 레이아웃 도입.

### ⑤ AI 기반 최적 여정 추천 (Smart Recommendation)
- **이유**: 최저가뿐만 아니라 '비행 시간/경유 횟수/가격'을 종합한 '최적'의 조합을 AI가 판단하여 제시.
- **기능**: "가장 가성비 좋은 선택" 태그 및 추천 사유 노출.

## 4. 단계별 개발 로드맵

### Phase 1: 기반 강화 및 기능 완성 (1~2개월)
- [ ] **필터 로직 완전 연동**: 백엔드 API와 연동하여 실시간 필터링(경유, 항공사, 가격) 완성.
- [ ] **임직원 인증 시스템 (Supabase Auth)**: 사내 계정 기반 로그인 및 권한 관리 구축.
- [ ] **다크 모드 디자인 최적화**: 전 페이지 다크 모드 일관성 및 시각적 보정.

### Phase 2: 사용자 경험 고도화 (3~4개월)
- [ ] **다구간 검색 기능 도입**: 검색 폼 및 결과 페이지 레이아웃 확장.
- [ ] **상세 정보 확인 및 외부 예약 연동**: 수하물 정보 표시 및 여행사/항공사 딥링크 구현.
- [ ] **모바일 UI 최적화**: 가격 그래프 및 필터 드로어 사용성 개선.

### Phase 3: Zero Maintenance & 지능화 (5~6개월)
- [ ] **가격 알림 시스템 구축**: Redis/Upstash 기반의 가격 변동 추적 및 알림 엔진.
- [ ] **AI 스마트 추천 엔진**: 사용자 검색 패턴 및 데이터 기반 최적 여정 추천.
- [ ] **자동 데이터 클렌징**: 만료되거나 오류가 있는 항공권 정보를 AI 에이전트가 자동 감지 및 수정.
